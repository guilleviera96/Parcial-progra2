./app
Backend del parcial. 
Incluye:
- Logica para agregar propietarios, agregar, eliminar, editar, listar mascotas.
- Esquema prisma con las entidades Propietarios y Mascotas.

./frontendparcial
Incluye:
- Interfaz de usuario basica con React Vite.
- El usuario debe registrar su documento para hacer el post y asi crearlo en la base de datos.
- Puede listar sus mascotas utilizando su DNI, en el cual sale una tabla con acciones (eliminar y editar).
